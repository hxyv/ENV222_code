  1. In an aerosol particle size distribution figure, the x-axis label is \(D_p(\mu m)\), and the y-axis label is \(\frac{\mathrm{d}N}{\mathrm{d}\log D_p}(\mathrm{cm}^{-3})\):

Write the R code for inserting the axis labels into a graph.

Code 
library(ggplot2)
library(latex2exp)

ggplot() +
    geom_blank() +
    theme_bw() +
    labs(y = TeX(r"($\frac{dN}{d\log D_p}$ $(cm^{-3})$)"),
         x = TeX(r"($D_p$ $(\mu m)$)"))
